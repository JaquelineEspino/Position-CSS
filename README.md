# Position-CSS
Repositorio de la lección 6 y 7 del módulo 2 del curso Frontend de **DEV.F** en donde se pone en práctica el tema de Position en CSS.

## Descripción
Este repositorio contiene una página HTML cuyo tema principal es sobre una serie animada llamada **"ARCANE"**. Se presenta la descripción y sinopsis de la serie, personajes relevantes y un formulario para unirse a la comunidad. También contiene una hoja de estilos para hacer la página web mas llamativa y que se vea mejor organizada.

## Características
En el HTML se utilizaron etiquetas como `<main>`, `<section>`, `<article>`, `<nav>`, `<h1>`, `<h2>`, `<h3>`, `<ol>`, `<footer>`, entre otras para darle una mejor estructura al contenido de la página web. En el archivo CSS, se utilizaron diferentes tipos de selectores (elementos, clase y ID) y diferentes atributos para darle un estilo único al sitio web, cambiando la tipografía, el color de fondo, el tamaño de la fuente, el tamaño de las imagenes agregadas, etc. También se utilizó la herencia poniendo atributos como font-family, text-align y margin en el body para aplicar estos atributos a todo el cuerpo de la página, utilizando el valor de inherit en margin para forzar la herencia. Se crearon tarjetas flexibles para la información de los personajes utilizando la técnica de Flexbox, haciendo que la información se vea mucho mas llamativa e interesante para el usuario. Se utilizan propiedades como flex-direction, align-items y gap para acomodar los elementos de las tarjetas y asignar espacio entre ellos. Se modifica el formulario convirtiendolo en un contenedor flex y se usan las mismas propiedades de las tarjetas de los personajes flex-direction y align-items con atributos column y center respectivamente, para alinear los elementos dentro del formulario. En este repositorio se agregan una barra de navegación al principio de la página web, un diseño para el footer donde se agrega información de contacto del desarrollador, y un mensaje de cookies al final de la página. Se usaron funciones como position: fixed y sticky para la barra de navegación y el mensaje de cookies para mantenerlos visibles en la pantalla, también se utilizó z-index para que se muestren por encima de los demás elementos.También se hace el uso de las media queries para hacer que la página web sea responsiva y se vea bien en cualquier dispositivo. Se agrega un menú "hamburguesa" para que aparezca cuando se este viendo la página en un dispositivo móvil.
